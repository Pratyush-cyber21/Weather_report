About ClimaTrack

Your personalized, interactive weather assistant ☀️⛅🌧️

🌐 What is ClimaTrack?

ClimaTrack is a multi-page, dynamic weather forecast platform built using:

  • ✅ OpenWeatherMap API for accurate forecasts
  • ✅ Chart.js for time-based visualizations
  • ✅ JavaScript for interactivity & data handling
  • ✅ Lottie animations for engaging visuals
  • ✅ HTML/CSS for structure and design

🧑‍💻 Developer Note

ClimaTrack was developed by Pratyush Sahoo as a full-stack responsive weather project for learning and showcasing modern frontend skills. The source code is hosted on GitHub and is continuously evolving.

📈 Future Plans

  • 🧭 Location-based weather using GPS
  • 🌍 5-day & 7-day detailed forecast view
  • 📊 More powerful data visualization & air quality insights
